# Habit Tracker

Проект для отслеживания привычек с возможностью интеграции с Telegram для рассылки напоминаний.

---

## Описание

Проект позволяет пользователям создавать, редактировать и удалять привычки, а также получать напоминания через Telegram. Проект реализован с использованием Django, Django REST Framework, Celery и PostgreSQL.

---

## Использование

1. Зарегистрируйся и авторизуйся через эндпоинты /api/users/register/ и /api/users/login/.
2. Используй эндпоинты для работы с привычками:
   - GET /api/habits/ — получить список своих привычек.
   - POST /api/habits/ — создать новую привычку.
   - GET /api/habits/public/ — получить список публичных привычек.
   - PUT /api/habits/<id>/ — редактировать привычку.
   - DELETE /api/habits/<id>/ — удалить привычку.
